# htmx-astro

The htmx documentation website, built with Astro.

## Repository Structure

- `/src/htmx.js` - The htmx library source (canonical version)
- `/www/` - The Astro documentation website
- `/www/public/js/` - Build artifacts, do not edit directly

## Context Files

Read the relevant `.claude/context/*.md` file before working in that area:

- `design.md` - when working on design, styling, layout, or UI components
- `docs.md` - when working on documentation content
- `gotchas.md` - when working on htmx versioning, the demo system, or morph navigation
