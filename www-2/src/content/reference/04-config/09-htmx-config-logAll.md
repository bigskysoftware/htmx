# **`htmx.config.logAll`**

When set to `true`, htmx will log all events to the console.

**Default:** `false`

## Example

```javascript
htmx.config.logAll = true;
```

```html
<meta name="htmx-config" content='{"logAll":true}'>
```

Useful for debugging htmx behavior during development.
