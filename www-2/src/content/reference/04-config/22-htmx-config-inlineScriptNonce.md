---
title: "htmx.config.inlineScriptNonce"
description: "Nonce value added to inline scripts generated by htmx"
---

Sets the `nonce` attribute on inline `<script>` elements that htmx generates, for use with a Content Security Policy that requires script nonces.

**Default:** `''` (empty string — no nonce added)

## Example

```javascript
htmx.config.inlineScriptNonce = "abc123";
```

```html
<meta name="htmx-config" content='{"inlineScriptNonce":"abc123"}'>
```

The nonce value should match the one provided in your `Content-Security-Policy` header.

See also: [`htmx.config`](/reference/config/htmx-config)
